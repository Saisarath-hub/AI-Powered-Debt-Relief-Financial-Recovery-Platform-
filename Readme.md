# AI Powered Debt Relief and Financial Recovery Platform

## Overview

The AI Powered Debt Relief and Financial Recovery Platform is a full-stack web application that helps users analyze their financial condition and receive intelligent recommendations for debt management.

## Features

- User Login
- Financial Dashboard
- Monthly Income and Expense Tracking
- Debt Analysis
- AI-Based Financial Recommendations
- Financial Summary
- Charts and Data Visualization
- FastAPI Backend
- SQLite Database

## Technologies Used

### Frontend

- React.js
- Vite
- Axios
- Chart.js

### Backend

- FastAPI
- Python
- SQLAlchemy

### Database

- SQLite

## Installation

### Backend

```bash
cd backend
venv\Scripts\activate
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
