# Sprint Planning

## Project Title

AI Powered Debt Relief and Financial Recovery Platform

## Objective

The project is divided into multiple sprints to ensure systematic development and timely completion.

## Sprint 1 – Planning

Duration: Week 1

Activities:
- Project idea selection
- Requirement gathering
- Technology selection
- Literature survey

Deliverables:
- Problem Statement
- SRS Document

---

## Sprint 2 – Design

Duration: Week 2

Activities:
- Architecture Design
- Use Case Diagram
- Activity Diagram
- Database Design

Deliverables:
- Design Documents

---

## Sprint 3 – Development

Duration: Week 3

Activities:
- React Frontend Development
- FastAPI Backend Development
- Database Integration
- REST API Development

Deliverables:
- Working Web Application

---

## Sprint 4 – Testing & Deployment

Duration: Week 4

Activities:
- System Testing
- Bug Fixing
- Documentation
- GitHub Deployment

Deliverables:
- Final Project
- GitHub Repository
