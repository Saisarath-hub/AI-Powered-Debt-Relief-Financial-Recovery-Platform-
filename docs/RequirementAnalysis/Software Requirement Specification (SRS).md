# Software Requirement Specification (SRS)

## Project Title

AI Powered Debt Relief and Financial Recovery Platform

## Introduction

The AI Powered Debt Relief and Financial Recovery Platform is a web-based application that helps users analyze their financial condition and receive AI-based recommendations for effective debt management.

## Purpose

The purpose of this software is to assist users in making informed financial decisions by calculating financial metrics and providing intelligent suggestions.

## Scope

The system allows users to:

- Login to the application
- Enter financial details
- Analyze financial health
- Calculate debt-to-income ratio
- Estimate EMI
- Generate AI recommendations
- Display financial charts
- Store financial information in a database

## Users

- Individual Users
- Financial Advisors
- Students (Project Demonstration)

## Hardware Requirements

- Intel i3 Processor or above
- 4 GB RAM
- 10 GB Free Storage

## Software Requirements

- Windows 10/11
- Python 3.13
- Node.js
- React
- FastAPI
- SQLite
- VS Code
- Git

## Conclusion

The proposed software provides a complete solution for financial analysis and debt management using Artificial Intelligence.
